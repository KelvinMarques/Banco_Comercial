# Banco_Comercial

## Passo a Passo

```
pipienv install
cd ./banco_comercial

```

### Criar um admin

```
python manage.py createsuperuser
```

### Rodar o servidor

```
python manage.py runserver
```

*Log que o servidor iniciou com sucesso*
